# montana-node

Core node abstractions for Montana.

## Overview

This crate provides core types for node operation including:
- `NodeRole` - The role a node operates as (Sequencer, Validator, or Dual)
- `NodeStage` - The current operational stage (Syncing or Active)
- `SyncProgress` - Progress information during sync stage
- `NodeConfig` - Configuration for the node
